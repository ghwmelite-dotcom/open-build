# Open Build — Brand Assets

Brand marks for **Open Build** (a show by OHWPStudios).

## Files

| File | Use |
|---|---|
| `open-build-icon.svg` | Master icon — gold "O" ring on a Ghana-green gradient. |
| `open-build-icon-512.png` | **Telegram group photo** (512×512, circle-crops cleanly). Also avatars/app icon. |
| `open-build-icon-192.png` | Smaller icon use. |
| `open-build-logo-on-dark.svg` / `.png` | Horizontal wordmark for **dark** backgrounds (white text, transparent). |
| `open-build-logo-on-light.svg` / `.png` | Horizontal wordmark for **light** backgrounds (dark text, transparent). |
| `tg-channel.svg` / `.png` | **Telegram announcements channel** photo — gold bell (alerts). |
| `tg-group.svg` / `.png` | **Telegram suggestions group** photo — gold lightbulb (ideas). |
| `tg-bot.svg` / `.png` | **Telegram bot** photo — gold robot head. |
| `channel-banner.svg` / `.png` | 1600×900 channel welcome banner (pin it at the top of the channel). |
| `live-banner.svg` / `.png` | 1920×1080 live‑stream cover (scales to a 1280×720 YouTube thumbnail). |

## Colours (Ghana national palette)

- Green `#0a8f56` (deep `#006B3F`)
- Gold `#FCD116`
- Red `#CE1126`
- Near-black `#0a0e0c`

## Notes

- The hub favicon is this same icon, embedded inline in `index.html` as a data-URI SVG — these files are the standalone exports for places that need a real image (Telegram, social, etc.).
- The `.svg` files are the masters; PNGs can be re-rendered from them at any size.
